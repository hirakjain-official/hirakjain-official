## 🚀 What I'm Building

### [Zero Hour](https://zero-hour-website.vercel.app)
> AI-powered debugging education platform

A browser-based learning environment where an AI Socratic mentor watches you code in real-time and asks guiding questions — never gives you the answer. Built for developers who want to actually understand what they're writing.

- **Stack:** React + Vite · Express · AWS EC2 (Mumbai) · AWS Bedrock · Monaco Editor · Docker · PostgreSQL · PM2 · Nginx
- **Submitted to:** AWS Zero Hour Hackathon — Campus AI track
- **Live at:** [zero-hour-website.vercel.app](https://zero-hour-website.vercel.app)

---

### CertifyX
> AI-powered certification & developer assessment platform

Replacing shallow MCQ culture with real scenario-driven assessments. Built for teams and communities that actually want to verify skill depth.

---

### VulnScan AI *(stealth)*
> AI code vulnerability scanner for early-stage startups

5,000+ vulnerability scenarios sourced from OWASP, CWE, CVE. Scenario-driven detection, not LLM hallucination. Built with engineering rigor first.
